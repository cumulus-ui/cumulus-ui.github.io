---
title: Checkbox
description: A control that allows toggling between checked, unchecked, and indeterminate states.
tag: cs-checkbox
---

## Usage

```html
<cs-checkbox>Accept terms</cs-checkbox>
<cs-checkbox checked>Notifications enabled</cs-checkbox>
<cs-checkbox indeterminate>Select all</cs-checkbox>
```

## States

- Default (unchecked)
- Checked
- Indeterminate
- Disabled
- Read-only

## With Description

```html
<cs-checkbox description="We'll send you an email when something changes.">
  Email notifications
</cs-checkbox>
```
