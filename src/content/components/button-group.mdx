> **Note:** Items are set via the `items` JavaScript property. See usage below.

## Usage

```html
<cs-button-group></cs-button-group>

<script>
  const group = document.querySelector('cs-button-group');
  group.items = [
    { text: 'Copy', icon-name: 'copy', id: 'copy' },
    { text: 'Edit', icon-name: 'edit', id: 'edit' },
    { text: 'Delete', icon-name: 'remove', id: 'delete' },
  ];
</script>
```
