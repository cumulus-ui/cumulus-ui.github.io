> **Note:** The `steps` property is set via JavaScript. See usage below.

## Usage

```html
<cs-wizard></cs-wizard>

<script>
  const wizard = document.querySelector('cs-wizard');
  wizard.steps = [
    {
      title: 'Choose engine',
      description: 'Select a database engine.',
      content: '<cs-container>Step 1 content</cs-container>',
    },
    {
      title: 'Configure settings',
      content: '<cs-container>Step 2 content</cs-container>',
    },
    {
      title: 'Review and create',
      content: '<cs-container>Step 3 content</cs-container>',
    },
  ];
</script>
```
