---
title: Icon
description: Renders a built-in or custom SVG icon in various sizes and colors.
tag: cs-icon
---

## Usage

```html
<cs-icon name="settings"></cs-icon>
<cs-icon name="search" size="big"></cs-icon>
<cs-icon name="close" variant="error"></cs-icon>
```

## Sizes

- `small`
- `normal` (default)
- `big`
- `large`
