## Usage

```html

  Create resource <cs-hotspot></cs-hotspot>
</cs-button>

<!-- Use within annotation-context for tutorial flows -->
<cs-annotation-context>
  <cs-form-field label="Name">
    <cs-input></cs-input>
    <cs-hotspot></cs-hotspot>
  </cs-form-field>
</cs-annotation-context>
```
