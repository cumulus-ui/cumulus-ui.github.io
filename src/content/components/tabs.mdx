<script>{`
  const el = document.getElementById('demo-tabs');
  if (el) {
    el.tabs = [
      { label: 'Overview', id: 'overview', content: 'Overview tab content' },
      { label: 'Details', id: 'details', content: 'Details tab content' },
      { label: 'Settings', id: 'settings', disabled: true },
    ];
    el.activeTabId = 'overview';
  }
`}</script>

> **Note:** Tabs are set via the `tabs` JavaScript property. See usage below.

## Usage

```html
<cs-tabs></cs-tabs>

<script>
  const tabs = document.querySelector('cs-tabs');
  tabs.tabs = [
    { label: 'Overview', id: 'overview' },
    { label: 'Details', id: 'details' },
    { label: 'Settings', id: 'settings', disabled: true },
  ];
  tabs.activeTabId = 'overview';
</script>
```
