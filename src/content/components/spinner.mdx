---
title: Spinner
description: An animated loading indicator available in multiple sizes.
tag: cs-spinner
---

## Usage

```html
<cs-spinner></cs-spinner>
<cs-spinner size="big"></cs-spinner>
<cs-spinner size="large"></cs-spinner>
```

## Sizes

- `normal` (default) — 16px
- `big` — 32px
- `large` — 48px

## Variants

- `normal` (default)
- `disabled`
- `inverted` — for use on dark backgrounds
