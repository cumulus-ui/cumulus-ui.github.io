## Usage

```html
<cs-grid id="my-grid">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</cs-grid>
<script>
  document.getElementById('my-grid').gridDefinition = [
    { colspan: 4 }, { colspan: 4 }, { colspan: 4 }
  ];
</script>
```
